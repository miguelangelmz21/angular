# Angular
## Repository to working with Angular

* We need to have installed Angular CLI or install with npm. We'll start with the version 12. Consider using node 14 (nvm install 14.21.3 -> nvm use 14).
    **npm install -g @angular/cli@12**

* To create a new project call "my-store" for example:
    **ng new my-store**

* Navigate to the '**/my-store**' folder and run the project using npm:
    **ng g c components/img**

* Finally To launch the application: 
    **ng serve**
